# VANT Creative Collective Website

This represents the digital presence of VANT Creative Collective. Built with a modern, high-performance tech stack focusing on design aesthetics and user experience.

## Tech Stack

- **Framework:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn UI
- **Animations:** Tailwind Animate & Custom CSS
